# 墨阅 (Moyue) —— 桌面端沉浸式阅读引擎

基于 Android 开源阅读器 Legado 重构的跨平台桌面版本。  
**保持书源兼容性，提供原生桌面体验。**

## 特性
- 📚 **书架管理**：增删改查、分组、封面、阅读进度
- 🔍 **书源系统**：导入/导出/编辑/调试，兼容 Legado 书源格式（约 98% 兼容）
- 📖 **阅读器**：4 种翻页模式、字体/主题设置、目录导航、书签
- 🛡️ **三级安全沙箱**：标准/兼容/信任模式，平衡安全与兼容
- 🔊 **TTS 朗读**：系统 TTS + 在线 TTS
- 🔄 **WebDAV 同步**：备份/恢复
- 📄 **本地书籍**：TXT/EPUB 导入，自动分章
